**Deliverable-Driven Project & Time Management for Architects**

Welcome to the full documentation for *Architect Project Planner*, a deliverable-centric project management methodology and software specification designed specifically for architects and small design studios.

## Purpose
This documentation serves as:
- A product and investor reference
- A technical specification for developers
- A methodology and workflow guide for architects
- A foundation for future SaaS or no-code implementations

## Core Principles
- **Deliverables drive planning**, not tasks
- **Weekly capacity planning**, not daily micromanagement
- **Realistic time estimation**, including iteration and coordination
- **Multi-authority submissions** treated as first-class entities
- **Lightweight workflows** with minimal administrative overhead
